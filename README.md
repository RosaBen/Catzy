# Catzy

**Catzy** is an online store dedicated to selling kitten photos. This project aims to offer a simple and aesthetic experience for cat lovers, whether they are buyers or just visitors.

## Features

- Gallery of kitten photos sorted by themes
- Account creation
- Shopping cart and secure payment system
- Smart search by breed, color, vibe, or emotion
- Filter and sort system
- Add to favorites
- User data modification, export, and deletion
